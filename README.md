# OPEN Opportunity × ACoolCOLLECTOR × Collectors Co-op

A launch-ready prototype for the OPEN LA retail program: branded landing page, product tiers, QR pack passports, compliant-by-default community campaign entry, printable collateral, partner command center, and Square/QuickBooks catalog mappings.

## Included

- Public launch page and Series 001 pricing
- Standard ($29.99), Premium ($49.99), and Chase ($79.99) pack tiers
- QR-driven pack passports and campaign entry
- Campaign disabled until Official Rules approval
- Super-admin partner dashboard for Stephen Smalls and Keith McPherson
- CSV export fields for Square Item ID and QuickBooks Item ID
- Printable 4 × 6 entry card, packaging sheet, social SVG, and brand guide
- Operating, security, and deployment documentation

## Run

```bash
npm install
cp .env.example .env
npm test
npm start
```

Open `http://localhost:3000`. See [deployment](docs/DEPLOYMENT.md), [security](docs/SECURITY.md), and [operations](docs/OPERATIONS.md).

## Important status

This is a secure-minded prototype, not a live sweepstakes or payment processor. The promotional entry endpoint is disabled by default. Square and QuickBooks credentials are placeholders only. Production launch requires counsel/accountant review, approved Official Rules, identity-provider MFA, managed encrypted storage, webhook verification, and partner approval.
